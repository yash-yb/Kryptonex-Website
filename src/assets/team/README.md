 *
 * Naming rules:

 *   - Underscores instead of spaces
 *   - Full name where possible
 *

 *   karan_choudhary.jpg
 *   sujal_patil.jpg
 *   events_head.jpg
 *   tech_member_01.jpg

 * After adding a file, import it in src/data/teamData.ts — NOT in TeamCard or TeamPage.
 *
 * Usage in teamData.ts:
 *   import karanPhoto from '../assets/team/karan_choudhary.jpg'
 *   ...
 *   { name: 'Karan Choudhary', role: 'Chairperson', photo: karanPhoto, ... }
 *
 * The TeamCard component will automatically use the photo when provided,
 * or fall back to a generated avatar when photo is undefined.
